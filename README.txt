BepinexPMLBridge v1.0.0
A BepInEx plugin to run PML mods without installing Pulsar Mod Loader.
Installation:
Install BepInEx into PULSARLostColony/.

Copy BepinexPMLBridge.dll to BepInEx/plugins/.

Copy PMLBridgeLibs/ to BepInEx/plugins/.

Place PML mods (e.g., LoadingScreenSkipper.dll) in PULSARLostColony/Mods/.

Notes:
Works with Thunderstore BepInEx mods in BepInEx/plugins/.
